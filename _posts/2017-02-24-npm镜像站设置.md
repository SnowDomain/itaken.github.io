切换 npm **镜像站点**, 方法有二

\1. 临时指定镜像源:

```shell
$ npm install --registry http://registry.cnpmjs.org express # 从中国镜像安装express
```

\2. 永久设置:

```shell
$ npm config set registry http://registry.cnpmjs.org  # 设置为中国镜像
$ npm install express
```
