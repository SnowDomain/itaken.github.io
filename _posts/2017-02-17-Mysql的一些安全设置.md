\1. 设置编码格式

```
[mysql]
default-character-set=utf8
```

\2. 安全更新, 防止全表更新
```sql
SET sql_safe_updates=1;
```

![safe update]({{ site.url }}/assets/images/201702/17-01.png)

```sql
SET sql_safe_updates=1, sql_select_limit=500, max_join_size=1000000;
```
